# Разведочный анализ CIFAR-10

## Состав датасета

- Строк в `trainLabels.csv`: 50000
- Изображений в папке `train`: 50000
- Строк в `sampleSubmission.csv`: 300000
- Изображений в папке `test`: 300000
- Отсутствующих train-файлов: 0
- Отсутствующих test-файлов: 0

## Классы

- Количество классов: 10
- Минимальный размер класса: 5000
- Максимальный размер класса: 5000
- Распределение классов сбалансировано: все классы представлены примерно одинаково.

## Изображения

- Все проверенные изображения имеют размер 32x32 и RGB-представление.
- Для анализа пиксельных признаков использована выборка из 2000 изображений.
- Самый темный класс по средней яркости: `frog`.
- Самый светлый класс по средней яркости: `airplane`.
- Самый низкий средний контраст: `deer`.
- Самый высокий средний контраст: `truck`.
